# Oscilator cu Punte Wien - 14 kHz

Proiect complet de electronică analogică realizat în cadrul Facultății de Electronică, Telecomunicații și Tehnologia Informațiilor. Proiectul cuprinde calculul teoretic, simularea PSpice, proiectarea layout-ului PCB și realizarea fizică a prototipului.

## Specificații Tehnice 
* **Frecvență de oscilație:** $f = 14$ kHz.
* **Amplitudine ieșire (Vpp):** 6.67 V.
* **Sarcina la ieșire:** 700 Ω.
* **Componente active:** Tranzistoare bipolare BC846B (SMD).
* **Alimentare:** Sursă simplă de tensiune (23.91 V).
